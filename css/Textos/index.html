Propriedade Text-transform

É responsável por definir quais caracteres vão estar em Maiusculo,minusculo, etc.

Capitalize coloca todas as palavras em maiusculo no começo.

uppercase coloca todas as palavras em caixa alta.

lowercase coloca todas as palavras em minusculo

Propriedade letter-spacing

Adiciona espaçamento entre cada caractere do texto

Lorem ipsum dolor sit amet consectetur adipisicing elit. Architecto labore, deserunt in natus facilis harum sit nostrum qui et ab dicta perferendis beatae quo voluptate iste eum laudantium autem dolore?


Propriedade word-spacing

Adiciona espaçamento entre cada palavra do texto



Propriedade white-space

Define como os espaços em branco do texto de um determinado elemento pe tratado.



Propriedade word-wrap

Define caso uma palavra seja muito linear-gradiente para caber na linha, se ela serpa quebrada ou toda palavra deve ficar na mesma linha.



Propriedade writing-mode

Define a orientação do texto ( vertical ou horizontal)

Idiomas que são lidas da direita para esquerda, como coreano e chines.

Propriedade test-overflow

Define como vamos mostrar ao usuario que há mais conteudo no texto alem do que ele pode visualizar naquele momento.
