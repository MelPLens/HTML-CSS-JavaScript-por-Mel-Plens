O que é CSS

Cascading Style Sheets, ou folha de estilo em cascata, mecanismo de adicionar estilos no documento HTML.

Criado por Hakon Wium Lie para facilitar a programação de sites.

Lembrete

  • NÃO é uma linguagem de programação
  • NÃO é uma linguagem de marcação
  • e SIM uma linguagem de estilo
